Installing SWAMP
----------------

Before installing SWAMP, make sure that you systems meets the following dependencies:
    * CCP4 7.1
    * Biopython 1.74
    * pyjob 0.4
    * conkit 0.11
    * biopython 1.74


1. Clone the repository
^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: shell

    git clone https://github.com/rigdenlab/SWAMP

2. Build SWAMP
^^^^^^^^^^^^^^

.. code-block:: shell

    cd SWAMP/build
    cmake ..
    make install
